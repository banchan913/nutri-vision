Nutri-Vision について

写真1枚からAIがカロリー・栄養素を自動解析し、日々の健康・ダイエットをサポートする次世代の栄養トラッキングアプリです。

毎日の食事の写真を撮るだけで、目標カロリーに達しているか、PFCバランスが適正に保たれているかがグラフで一目でわかります。面倒な手入力が不要になり、あなたの健康的な選択を後押しします。

使い始めるための設定

  • 1. プロフィール: あなた専用の目標カロリーを算出するための基本情報です。
  • 2. AI解析設定: 食事を自動解析するためのGemini APIキーを設定します（取得無料）。
  • 3. クラウド同期 (任意): スマホ単体で使用する場合は設定不要です。PCと同期したい場合のみ設定してください。

2. AI解析の設定

Gemini APIの利用には専用のキーが必要です。
Google AI Studio (こちらをクリック) で無料で取得できます。

詳しい取得方法を開く
  1. 上のリンクから Google AI Studio にGoogleアカウントでログインします。
  2. 画面左上の 「Get API key」 をクリックします。
  3. 「Create API key」 ボタンを押し、キーを作成します。
  4. 画面に表示された AIza... から始まる文字列をコピーし、下の入力欄に貼り付けてください。

3. クラウド同期（Google スプレッドシート）

☁️ クラウド同期（Google スプレッドシート）の設定手順

この設定を行うと、スマホやPCなど複数の端末でデータを共有したり、大切な記録を永続的にバックアップできるようになります。

  1. スプレッドシートの準備: 新しいスプレッドシートを作成します。
  2. Apps Scriptを開く: 上部メニューの「拡張機能」から「Apps Script」をクリックします。
  3. コードの貼り付け: 下の「GASスクリプトをコピー」ボタンを押し、エディタに元からあるコードをすべて消して貼り付けます。
  4. デプロイ（公開）: 右上の「デプロイ」>「新しいデプロイ」を選択。
    • 種類の選択（歯車アイコン）:「ウェブアプリ」
    • 次のユーザーとして実行:「自分」
    • アクセスできるユーザー:「全員」
  5. 認証: 「アクセスを承認」を押し、自分のGoogleアカウントを選択。警告が出た場合は「詳細」>「Nutri-Vision（安全ではないページ）に移動」をクリックします。
  6. URLのコピー: 表示された「ウェブアプリのURL」をコピーして、以下の欄に貼り付けてください。
同期ボタンについて: 右上の「同期」ボタンを押すと、スプレッドシートに保存されている最新のデータを読み込み、現在の画面に反映させます（端末間のデータ消失を防ぎます）。
※スマホ単体でご利用の場合はこの設定は不要です。データは自動で端末内に保存されます。
⚠️ 重要: GASの「デプロイ」設定で、アクセスできるユーザーを「全員（Anyone）」にする必要があります。「自分のみ」では通信エラー（CORS）が発生します。
同期できない（403エラー等）場合
  • 設定確認: 「新しいデプロイ」でアクセスできるユーザーを「全員」に設定して公開していますか？
  • アカウント競合: 複数のGoogleアカウントにログインしている場合、エラーになることがあります。シークレットブラウザでお試しください。
  • URL確認: `/dev` で終わるテスト用URLではなく、`/exec` で終わる公開用URLを使用していますか？
  • 最終手段: 以前のGASコードをすべて消し、最新版を貼り付けて再デプロイしてください。

スマホへ設定を転送

スマホにGASのURLを打ち込むのは大変です。このボタンでQRコードを表示し、スマホのカメラで読み取るだけで設定を引き継げます。
